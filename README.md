# 图片压缩工具 (Image Compressor)

## 版本信息
- 版本号：1.0.0
- 创建日期：2024-03-21
- 最后更新：2024-03-21

## 项目描述
这是一个基于网页的图片压缩工具，采用苹果风格设计，提供简洁优雅的用户界面和流畅的操作体验。

### 主要功能
1. 图片上传
   - 支持 PNG、JPG 等常见图片格式
   - 支持拖拽上传和点击上传
   - 实时预览上传的图片

2. 图片压缩
   - 可调节压缩比例
   - 实时预览压缩效果
   - 显示压缩前后的文件大小对比

3. 图片下载
   - 一键下载压缩后的图片
   - 保持原始图片格式

## 技术栈
- HTML5
- CSS3
- JavaScript (原生)
- 使用 browser-image-compression 库进行图片压缩

## 页面结构
### 主页面 (index.html)
1. 顶部区域
   - 标题
   - 简要说明文字

2. 上传区域
   - 拖拽上传区域
   - 文件选择按钮

3. 预览和控制区域
   - 原图预览
   - 压缩设置控制面板
   - 压缩后图片预览
   - 文件大小信息显示

4. 操作区域
   - 下载按钮
   - 重新上传按钮
